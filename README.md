# Pricing-Card
A responsive pricing card UI built with Tailwind CSS as a practice project for learning modern utility-first frontend styling.

# 💳 Pricing Card UI – Tailwind CSS Practice Project

This project is a simple and responsive **Pricing Card UI** created while learning **Tailwind CSS**.
It demonstrates the use of utility-first classes to build modern and clean user interfaces quickly.

---

## 🚀 Features

* 🎨 Modern card design
* 📱 Responsive centered layout
* ✨ Hover button effects
* 🧾 Pricing plan section
* ⚡ Built entirely using Tailwind CSS utility classes

---

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS

---

## ⚙️ What I Practiced

* Flexbox alignment using Tailwind
* Spacing utilities (`p`, `m`, `gap`)
* Typography styling
* Colors and backgrounds
* Rounded corners and shadows
* Hover transitions and button effects
* Responsive layout basics

---

## ▶️ How to Run

1. Download or clone the repository
2. Make sure Tailwind CSS is properly compiled
3. Open `Pricing-card.html` in your browser
